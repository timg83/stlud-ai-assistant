# ADR 0005: Use React and Vite for the Widget

## Status

Accepted

## Context

Voor de widget is gekozen voor React met Vite en TypeScript.

## Decision

We bouwen de chatwidget als React/Vite-applicatie die kan worden gebundeld voor embedding.

## Consequences

- Snelle ontwikkelcyclus en breed ecosysteem.
- Build pipeline vereist Node.js tooling.
