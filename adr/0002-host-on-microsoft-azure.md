# ADR 0002: Host on Microsoft Azure

## Status

Accepted

## Context

Er is een expliciete voorkeur voor een Microsoft/Azure stack.

## Decision

We standaardiseren de MVP op Azure-diensten voor hosting, AI, storage, monitoring en identity.

## Consequences

- Goede aansluiting op Microsoft 365 en toekomstige SharePoint-integraties.
- Leverancierafhankelijkheid richting Azure.
