# Terraform Skeleton

Deze map bevat de startstructuur voor Azure-infrastructuur. Terraform is in de huidige container niet beschikbaar, dus de code is nog niet gevalideerd.

## Beoogde resources

- Resource group
- App Service plan en Web App voor backend
- Cosmos DB account en database
- Storage account en blob container
- Azure AI Search
- Azure OpenAI koppeling via configuratie
- Key Vault
- Application Insights
