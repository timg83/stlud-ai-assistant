# ADR 0012: Use Public Anonymous Chat with Rate Limiting

## Status

Accepted

## Context

Voor MVP is gekozen voor een publieke anonieme chat.

## Decision

We bieden anonieme toegang tot de chat en beschermen de API met rate limiting en inputrestricties.

## Consequences

- Lagere frictie voor ouders en leerlingen.
- Hogere noodzaak voor abuse-preventie en monitoring.
