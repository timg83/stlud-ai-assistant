# ADR 0003: Embed Chat Widget on Existing Website

## Status

Accepted

## Context

De assistent moet via de bestaande schoolwebsite beschikbaar zijn.

## Decision

We leveren de gebruikersinterface als in te sluiten chatwidget in plaats van als losstaande applicatie.

## Consequences

- Lage adoptiedrempel voor eindgebruikers.
- Rekening houden met CORS, theming en embed-beperkingen van de bestaande site.
